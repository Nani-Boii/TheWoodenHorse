 # The Wooden Horse

 A 2D platformer where the player navigates mythological traps using physics-based mechanics. As the payer progess they're tasked to collect 4 wooden pieces for the wooden horse be built.

 # Built With
* [Unity 6.3] - Game Engine
* [C#] - Programming Language
